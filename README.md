# Freelance Company — Fullstack проект

Короткое описание
- **Freelance Company** — готовая к демонстрации платформа для поиска и выполнения сервисных услуг. Платформа объединяет заказчиков и исполнителей, предоставляя удобный интерфейс для публикации услуг, оформления заказов, безопасных транзакций и мгновенного чата.

О чём проект
- Проект моделирует реальный процесс фриланс-платформы: исполнители публикуют услуги, заказчики выбирают и оформляют заказы, средства резервируются и переводятся через модуль транзакций, а после выполнения заказов оставляются отзывы. Встроенный WebSocket-чат обеспечивает общение в реальном времени между участниками сделки.

Преимущества проекта
- Готовая архитектура «frontend + backend», легко расширяемая и разворачиваемая.
- Продуманная модель данных для услуг, заказов, транзакций и отзывов.
- Реальное поведение платформы: безопасность платежей, статусный цикл заказов, система рейтингов и отзывов.
- WebSocket-чат для мгновенного обмена сообщениями и улучшенной коммуникации.
- Низкий порог входа: по умолчанию используется SQLite, что позволяет поднять проект локально за несколько минут.

Что происходит в проекте (короткий flow)
- Исполнитель создаёт услугу с описанием, ценой и сроками.
- Заказчик находит услугу и оформляет заказ — создаётся запись заказа и инициируется транзакция.
- При подтверждении оплаты средства резервируются (модель транзакций) и перечисляются исполнителю по выполнении.
- В процессе сделки участники общаются через WebSocket-чат (`/ws/{user_id}`).
- По завершении заказчик оставляет отзыв — формируется рейтинг исполнителя.

Кому полезен
- Демонстрации продукта, быстрым прототипам маркетплейса услуг, учебным целям и тестированию интеграций (платежи, уведомления, масштабирование).

Быстрый старт: как установить зависимости и запустить backend (dev)

Установка зависимостей (backend)

```bash
cd backend
# создать виртуальное окружение
python -m venv .venv
.venv\Scripts\activate    # Windows
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

Запуск dev-сервера (FastAPI + Uvicorn)

```bash
cd backend
.venv\Scripts\activate    # если ещё не активировано (Windows)
fastapi dev
```

После запуска API будет доступно по адресу: `http://127.0.0.1:8000`.

Установка зависимостей и запуск frontend 

```bash
cd frontend
npm install
npm run dev
```

По умолчанию frontend использует API по `http://127.0.0.1:8000/api` (см. `frontend/src/lib/api/axios.ts`).

Переменные окружения (кратко)
- `SECRET_KEY` — секрет для токенов (backend/config/auth.py).
- `NAME_DB` — опционально, имя БД; по умолчанию используется `sqlite:///./app.db`.
- `NEXT_PUBLIC_API_URL` — (frontend) укажите, если API доступен по другому адресу.

API — краткий обзор
- Все роуты находятся под префиксом `/api`.
- Ключевые префиксы: `/api/auth`, `/api/users`, `/api/services`, `/api/orders`, `/api/transactions`, `/api/reviews`, `/api/specializations`, `/api/messages`.

WebSocket
- Endpoint: `/ws/{user_id}` — используется для обмена сообщениями в реальном времени между пользователями.

Дополнительно
- Статические изображения — `backend/images/`, endpoint `GET /{image_name}` отдаёт файл.
- Миграции: `alembic upgrade head` в папке `backend`.
